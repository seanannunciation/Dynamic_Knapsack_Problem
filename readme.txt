Steps to be followed for compilation and Execution of assignmentP3.java

Step 1 : Please copy the file assignmentP3.java in your directory along with the makefile.txt from sannunc1_java_p3.tar.gz folder

Step 2 : It is important that there exists an input.txt file in the same directory where assignmentP3.java, makefile.txt is present. If it is not there then please create one named input.txt

Step 3 : Type the command make -f makefile.txt

Step 3 will  compile the code, Execute the code as well as print the output file named output.txt in the same directory

The OUTPUT is in the following format:
<PROBLEM SIZE> <MAXPROFIT> <TIME IN SECS>
<ITEMS IN THE KNAPSACK>
